This uses the AWS IP Ranges JSON file to determine if a given IP address is used by AWS and if so what service(s) and region(s) it is allocated to. This information is not particularly precise and the most common services that will be seen are CloudFront and EC2 (though there are other values!).

When a hostname is entered, rather than directly looking the value up in the JSON file, the Cloudflare DNS over HTTPS API is used to get the IPv4 and IPv6 addresses for the given domain. Only A and AAAA records will be checked.
